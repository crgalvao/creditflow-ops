# Diagrams

This folder contains Mermaid diagram source files for CreditFlow Ops.

## Files

- `architecture.mmd` — high-level system architecture.
- `application-submission-sequence.mmd` — main application submission sequence.
- `event-flow.mmd` — SNS/SQS event-driven workflow.
- `user-flow.mmd` — analyst-facing UI flow.

## Notes

These diagrams are intentionally lightweight drafts at this stage. They should be updated as implementation details become concrete.
